# Medication Reminder App using ChatGPT

A prompt-based application designed to help users manage their medications effectively using AI-powered interactions.

---

##  Introduction

<img width="400" height="300" alt="image" src="https://github.com/user-attachments/assets/e96c987e-bd9a-4737-aa94-2812038bc299" />


The advancement of large language models like ChatGPT has enabled the development of intelligent applications that interact naturally with users.

This project focuses on building a **Medication Reminder App** that uses prompt-based input to help users remember and track their medications efficiently.

---

## Objectives

* Develop a prompt-based AI application
* Improve medication adherence
* Enable natural language interaction
* Solve real-world healthcare problems using AI

---

## Problem Statement

Many people forget to take medications on time, especially:

* Elderly individuals
* Busy professionals
* Patients with multiple prescriptions

Missing doses can lead to serious health risks. This app provides **timely reminders and smart assistance**.

---

## System Overview

The system uses ChatGPT to:

* Process user prompts
* Store medication schedules
* Generate reminders
* Answer user queries

---

## Architecture

```
User Input (Prompt)
        ↓
Frontend (App UI)
        ↓
ChatGPT API (Prompt Processing)
        ↓
Scheduler System
        ↓
Notification Service
```

---

## Working Principle

1. User enters medication details (name, time, dosage)
2. AI processes the input
3. Reminder is scheduled
4. Notifications are sent
5. User can interact using prompts

---

##  Sample Prompts

```
Remind me to take Paracetamol at 8 AM daily
Did I take my medicine today?
Show my medication schedule
```
